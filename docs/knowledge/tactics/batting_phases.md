# Batting phases in T20 (Powerplay → Middle → Death)

This doc explains common T20 batting priorities by phase. These are patterns, not rules.

## Powerplay (overs 1–6)

**Goal:** maximize boundary opportunities while not losing too many wickets.

Typical traits:
- higher boundary % than later phases
- more risk-taking
- value of a wicket is high because middle overs become harder if you lose too many early

Common approaches:
- attack weaker bowlers
- target straight boundaries if swing/seam is on offer
- keep strike rotation to avoid dot-ball pressure

## Middle overs (overs 7–15)

**Goal:** maintain run rate and set up a launch at the death.

Typical traits:
- more spin
- field spreads out; boundaries are harder
- matchups matter (left-right, batter vs spinner types)

Common approaches:
- prioritize singles and twos
- pick moments to attack (short boundary, weaker matchup)
- avoid collapses; wickets here can stall an innings

## Death overs (overs 16–20)

**Goal:** convert wickets-in-hand into maximum runs.

Typical traits:
- high boundary rate
- more yorkers, wide yorkers, slower balls
- mis-hits can still clear boundaries in many venues

Common approaches:
- premeditated shots (ramps, slog sweeps) depending on bowler type
- target matchups and field placements
- accept higher dismissal risk if required run rate is high
